Installation Guide

To enable XDG-URL, simply install the xdgurl tool from here: https://www.linux-apps.com/p/1136805/

After xdgurl is installed, just click the "Install" button on any supported fileformat on openDesktop.org.

Browser Setup

In Firefox a program selection window opens the first time you click on an Install button: Choose "xdgurl" as custom URL handler and tick "remember the file association".

If you don't get the program selection window or you are using another browser, please set the xdgurl program as custom URL handler for xdg:// scheme in the browser's settings.
